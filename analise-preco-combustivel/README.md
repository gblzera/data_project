# Análise de Vendas de Combustíveis no Brasil

Este projeto realiza uma análise dos volumes de vendas mensais de combustíveis no Brasil, usando dados públicos da ANP (2012 a 2025).

## Estrutura
- `data/`: CSV com dados brutos
- `src/`: Scripts de processamento e visualização
- `notebooks/`: Jupyter Notebook para análise interativa
- `outputs/`: Gráficos e arquivos gerados
